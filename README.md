# §AI — paragrafen.ai

> *"Fråga Paragrafen"*

En gratis svensk juridisk AI-assistent riktad till allmänheten.

## Vad är §AI?

§AI hjälper dig förstå svensk rätt — lagar, förordningar, rättspraxis och förarbeten — på klarspråk. Tjänsten är gratis, öppen källkod och självhostad.

## Täckta rättsområden

§AI täcker svensk civilrätt, avtalsrätt, arbetsrätt, hyresrätt, konsumenträtt, förvaltningsrätt m.fl.

**Ej täckta områden** (se [DISCLAIMER.md](DISCLAIMER.md)):
- Straffrätt
- Asylrätt och migration
- Skatterätt
- Vårdnad, boende och umgänge (VBU-tvister)
